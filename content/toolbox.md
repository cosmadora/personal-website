These are tools I frequently use in my development journey.

## Golang

### CLI App Manager

- [Kong](https://github.com/alecthomas/kong) [`go get github.com/alecthomas/kong`]: command-line parser for Go that supports complex command-line structures with minimal developer effort

### Config Management

- [koanf](github.com/knadh/koanf) : Can do combined ENV, yaml, toml, etc. Very powerful. Alternative to `spf13/cobra`
- [env](https://github.com/caarlos0/env) [`go get github.com/caarlos0/env/v11`]: A simple, zero-dependencies library to parse environment variables into structs

### Database Management

- [goose](https://github.com/pressly/goose) [`go get github.com/pressly/goose/v3`]: A database migration tool. Supports SQL migrations and Go functions.
- [pq](https://github.com/lib/pq) [`go get github.com/lib/pq`]: Pure Go Postgres driver for database/sql

### HTTP Routers

- [Gin](https://github.com/gin-gonic/gin) [`go get github.com/gin-gonic/gin`]: Super fast HTTP web framework ([Docs](https://gin-gonic.com/docs/))
